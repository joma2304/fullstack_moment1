Bootstrap

Denna delen av webbplatsen använder Bootstrap för stylingen.

Fördelarna med detta:

  • Det går snabbare att styla webbplatsen och göra den enhetlig.
  • Det finns färdig lösning för responsivitet exempelvis vid navigationsmenyn jag använde.
  • Personligen så tycker jag att Bootstrap är lite mer snabb-använt än exempelvis Tailwind eftersom det känns som att det är mer stylingn i klasserna överlag, därav behöver man inte använda lika många eller på lika många element.

Nackdelar med detta:

  • Webbplatsen löper risk för att bli lik många andra webbplatser som använder sig av Bootstrap som css ramverk.
  • Jag känner att jag inte har samma frihet som när det kommer till vanlig CSS eller SCSS/SASS.
