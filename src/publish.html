Publicering

Uppgiften har publicerats via Netlify. Detta har gjorts eftersom att det finns ett kostnadsfritt alternativ som är snabbt och smidigt. Jag har i tidigare kurser använt mig av Netlify tillsammans med exempelvis Render för min webbtjänst. Jag tänker att jag i kommande uppgifter ska ge mig på att testa någon annat ( ex. vercel eller azure) för att bekanta mig med något nytt.

Netlify stödjer statiska webbplatser byggda med HTML, CSS och JavaScript. Det finns även stöd för språk som Node.js via serverlösa funktioner (Netlify funtions) för att exempelvis göra API-anrop. Språk som Ruby eller PHP fungerar endast ifall de omvanlas till statiska filer (eller med externa lösningar, exempelvis med Render för hosting av webbtjänst) (klicka här för att se mer). Det stödjer även ramverk så som Angular, React, Vite och fler.

Det går att publicera sin webbplats direkt från ens Githubrepo vilket jag gillar starkt. Vid en ny push till sitt repo uppdateras även webbplatsen direkt utifrån den nya koden.
